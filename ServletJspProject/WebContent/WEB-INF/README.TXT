Le répertoire WEB-INF contient toutes les informations et fichiers (JSP, librairies) dont on a besoin dans un application 
y-compris les classes compilées (WEB-INF/classes) quand on déploie l'application sur un serveur. Dans un IDE il peut en être autrement, 
les classes compilées sont déployées dans un répertoire de travail local au serveur.
